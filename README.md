# ArcadeGame

## Description

This is a simple Arcade Game i've built for myself using lwjgl, an OpenGL library for java development. This is only a POC and mainly for fun.

### Install maven dependencies

```sh
$ mvn clean install
```

## Compile

```sh
$ mvn compile
```

### Run
```sh
$ mvn exec:java
```
